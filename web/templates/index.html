> ABOUT

BOUNTIFUL IS A BUG BOUNTY PLATFORM FOR THE FE COMPILER.

EACH CHALLENGE IS A 15-PUZZLE DEPLOYED ON-CHAIN IN AN UNSOLVABLE STATE. IF YOU CAN SOLVE IT, YOU HAVE FOUND A COMPILER BUG.

> HOW TO PARTICIPATE

  1. FIND A BUG IN THE FE COMPILER
  2. LOCK A CHALLENGE ON-CHAIN
  3. EXPLOIT THE BUG TO SOLVE THE PUZZLE
  4. CLAIM YOUR BOUNTY
